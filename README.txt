【项目简介】
  本项目是一套面向软件工程专业学生的就业赋能资料中心，
  由 211 软件工程学院教师用于课堂授课与学生自主查阅。
  内容涵盖：简历写作方法论、简历优化对比、优秀简历范本、
  自助优化工具、大厂高频面试题集等。

  所有资料均为 HTML 网页，双击即可用浏览器打开，
  无需安装任何软件，也无需联网。


================================================================
  一、如何使用
================================================================

  1. 打开入口：
     双击【index.html】，即可进入资料导航首页。

  2. 浏览资料：
     首页以卡片形式展示全部资料，点击任意卡片
     即可打开对应内容。顶部搜索框支持按标题 / 分类
     实时筛选。

  3. 推荐阅读路径（从认知到实操）：
     认知方法 → 优化对照 → 自助动手 → 面试准备 → 对照范本

================================================================
  三、如何新增资料到导航页
================================================================

  导航页采用"数据驱动"设计，新增资料非常简单：

  1. 把新的 .html 文件放到本项目文件夹（与 index.html 同级）。

  2. 用记事本 / 编辑器打开 index.html，找到下面这段：

         const LINKS = [
             ...
         ];

  3. 照着已有的格式，在数组里追加一项即可：

         {
           title: "资料标题",
           file:  "文件名.html",
           desc:  "一句话简介",
           cat:   "简历指导",      // 分类见下
           icon:  "📄",            // 任意 emoji 图标
           color: "blue",          // blue/pur/green/amber
           diff:  "标签文字"        // 可选，如"必看""范本01"
         }

  4. 保存后刷新 index.html 即可看到新卡片，无需改动布局。

  可用分类(cat)：简历指导 / 面试准备 / 工具自助 / 简历示例
  配色(color)：blue 蓝(指导) / pur 紫(面试) /
               green 绿(工具) / amber 琥珀(示例)


================================================================
  四、注意事项
================================================================

  1. 所有文件之间通过"相对路径"互相引用，
     请勿单独移动某个文件，如需搬移请整个文件夹一起移动。

  2. 文件夹内的 .workbuddy 目录用于存放项目数据，
     请勿删除。

  3. 范本页面(优秀简历示例)右下角有"打印"按钮，
     可直接打印或另存为 PDF 分发给学生。
